🔐 Sistema de Login
Login e cadastro feito com HTML, CSS e JavaScript.

✨ Funcionalidades

🔑 Login com validação de e-mail e senha
📝 Cadastro de novos usuários
💪 Indicador de força da senha em tempo real
💾 Armazenamento local com localStorage
✅ Validação de campos com mensagens de erro
🌙 Tema escuro moderno e responsivo
🔔 Notificações

📁 Estrutura do Projeto
sistema-login/
index.html    # Estrutura da página
style.css     # Estilos e tema escuro
script.js     # Lógica de login e cadastro
README.md     # Documentação

🛠️ Tecnologias Utilizadas
HTML5 - Estrutura da página
CSS3 - Estilização e animações
JavaScript - Lógica e validações
localStorage - Armazenamento dos usuários
